# Сложно сосредоточиться

Проектная работа 3-го спринта курса "Веб-разработчик" от Яндекс Практикум.

## Описание проекта

Адаптивный одностраничный сайт с поддержкой светлой и тёмной темы оформления. Проект посвящён теме концентрации внимания и способам её улучшения.

## Технологии

- HTML5
- CSS3
- Адаптивная вёрстка (Mobile First)
- CSS Grid
- Flexbox
- CSS Custom Properties (переменные)
- Логические CSS-свойства
- Автоматическое переключение темы через `prefers-color-scheme`

## Особенности

- Три варианта отображения: мобильная версия (375px), планшет (768px), десктоп (1024px+)
- Поддержка светлой и тёмной темы с автоматическим и ручным переключением
- Семантическая разметка
- "Ленивая" загрузка изображений
